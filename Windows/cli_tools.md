# 🧰 Dev Tools Setup (Windows + Scoop)

## 📦 Ferramentas CLI

### 🔎 Pesquisa / Navegação

- **fzf** — fuzzy finder interativo

- **fd** — alternativa moderna ao `find`

- **ripgrep (rg)** — pesquisa ultra-rápida em ficheiros

- **zoxide** — `cd` inteligente (aprende diretórios)

### 📂 File Managers / Editores

- **nvim** — Neovim (editor principal)

- **micro** — editor simples tipo nano melhorado

- **nano** — editor clássico

- **spf** — Superfile (file manager no terminal)

### 📊 Monitorização / Sistema

- **btop** — monitor de sistema moderno

- **bottom (btm)** — alternativa ao top

- **fastfetch** — info do sistema (rápido)

- **neofetch** — info do sistema (clássico)

- **duf** — uso de disco moderno
