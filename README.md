# Ricing_Repo
A repository with my configurations on both operating systems (Windows + Linux), also including a list of useful applications I recommend, as well as some scripts for switching themes. I configured Windows as closely as possible to Linux, with theme switching for the terminal, VS Code, Nilesoft Shell, Komorebic, and others.

## 🎨 Media (Criação e Edição)

Ferramentas para edição de imagem, som e vídeo.

- **Krita** — pintura digital e ilustração profissional.
- **Vegas** — editor de vídeo avançado (edição não-linear).
- **Inkscape** — editor de gráficos vetoriais (alternativa ao Illustrator).

- **Audacity** — gravação e edição de áudio open-source.
- **Photoshop** — edição profissional de imagens.
- **Paint.net** — edição de imagens.

Ferramentas e ambientes para programação, bases de dados e produtividade geral.

- **ProtonVPN** — VPN segura e rápida.
- **MySQL** — sistema de gestão de bases de dados relacional.
- **MongoDB** — base de dados NoSQL orientada a documentos.
- **SQLite** — base de dados leve, local e sem servidor.

## ⚙️ Optimization (Performance e Manutenção)

Ferramentas para otimizar o desempenho, monitorizar e gerir o sistema.

- **NanaZip** — alternativa moderna ao 7-Zip (mesma base técnica).
- **PowerToys** — utilitários avançados oficiais da Microsoft.
- **Holdem** — ferramenta de otimização do sistema.
- **LatencyMon** — mede e diagnostica latência no sistema.
- **Razer Cortex** — gestor de desempenho para jogos.
- **Cheat Engine** — análise de memória (uso legítimo: debugging e testes).
- **Flameshot** — ferramenta avançada de capturas de ecrã.
- **DNS Bench** — benchmark de servidores DNS.
- **DefenderUI** — interface melhorada para o Windows Defender.
- **Recuva** — recuperação de ficheiros eliminados (duplicado — ver categoria “Work”).
- **Crap Cleaner / Fixer** — limpeza e otimização de ficheiros temporários.
- **Chris Titus Tool** — script de otimização e configuração do Windows.
- **WinToys** — utilitário gráfico para tweaks e performance.
- **Context Menu Manager** — gestão de itens no menu de contexto.
- **Shell Context Menu** — personalização do menu do botão direito.
- **Shutdown Timer Classic** — temporizador para desligar o PC.
- **TCP Optimizer** — otimização de parâmetros de rede TCP.
- **RAMMap** — análise detalhada de uso da memória.
- **DNS Benchmark** — medição e comparação de servidores DNS.
- **WinHance** — optimizações e instalações, é muito bom.
- **PC Manager** — optimizações e instalações, é muito bom.
- **ChrisTitusTool** — usa o scrip para tweaks e isso e podes usar o comando: $irm christitus.com/win | iex$ para instalar ferramentas e outras coisas, é bom.
- **O&O ShutUp10++** — desativa um monte de telemetria como o WinHance.
- **BleachBit** — faz limpeza.

---

## 🗂️ Notas

- Algumas ferramentas (ex. **Recuva**, **DNS Benchmark**) aparecem em mais de uma categoria por terem múltiplos usos.
- Várias são **open-source** e disponíveis tanto no **Windows** como no **Linux**.
- Para gestão e instalação rápida, considera usar:
  - **Winget** (Windows)
  - **Scoop** (Windows)
  - **APT / DNF / Pacman** (Linux)
