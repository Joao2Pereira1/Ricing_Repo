# 🧠 Hidden gems (Linux power user)

## 🔍 Pesquisa / navegação no sistema

- FSearch → tipo Everything (Windows), SUPER rápido
- fd → melhor que `find`
- ripgrep → melhor que `grep`

👉 combo fd + ripgrep = produtividade absurda

---

## 🧰 Terminal / CLI avançado

- btop → melhor monitor CLI
- zoxide → navegação inteligente (aprende hábitos)
- bat → `cat` com syntax highlight
- eza → `ls` moderno (ex-`exa`)

---

## 📦 Gestão / sistema (Arch vibes)

- paru → helper AUR (melhor que yay para muitos)
- stow → gerir dotfiles como um pro
- reflector → otimizar mirrors do pacman
